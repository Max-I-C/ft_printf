# ft_printf

This project is a simple implementation of the printf function from <stdio.h>.

At 42, this project serves as an introduction to variadic functions. This function can and should also be added to your libft for use in future projects.
## Format specifier implemented

| Format            | Value associated |
| ----------------- | ---------------- |
| %c | char |
| %s | string |
| %d | integer |
| %i | signed integer (decimal) |
| %u | unsigned int |
| %x & %X | hexadecimal value |
| %p | memory address  |


## Installation

- Type make
- Create a test program and include the header
- Compile the test file with libftprintf.a and execute the program

```bash
  make
  gcc test.c libftprintf.a -o prog
  ./prog
```
